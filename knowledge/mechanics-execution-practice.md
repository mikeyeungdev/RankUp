# Mechanics, Execution, And Practice

Use this note when the transcript discusses combos, skillshots, animation timing, camera control, cursor control, kiting, orb walking, smiting, dodging, or practice tool drills.

Core coaching model:
- Mechanics become coachable when they are tied to a repeatable situation and a measurable success condition.
- Separate decision error from execution error. A missed combo after a bad engage is not the main issue.
- Good mechanics training uses small reps: one combo, one spacing rule, one smite timing, one camera habit, or one dodge pattern.
- Speed is not always the goal. Some mechanics improve by delaying, holding an ability, waiting for enemy movement, or confirming setup.
- Practice tool drills should connect back to a game trigger so the student knows when to use the mechanic.

Review questions:
- Was the failed play caused by wrong decision, wrong setup, or poor execution?
- Which exact input, timing, or camera habit broke down?
- Was the student rushing a mechanic that should have been delayed?
- Can this be practiced in a controlled drill?

Training goals:
- Create a five-minute drill for the exact mechanic discussed in the transcript.
- Review three failed mechanical plays and label decision, setup, or execution.
- Practice one combo or movement pattern until the student can perform it cleanly several times in a row.
